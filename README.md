# Reception App

Flutter reception/check-in app backed by Supabase.

## Supabase Setup

1. Create a Supabase project from the [Supabase dashboard](https://supabase.com/dashboard).
2. Copy `.env.example` to `.env`.
3. Set:
   - `SUPABASE_URL` from **Project Settings -> API -> Project URL**
   - `SUPABASE_ANON_KEY` from **Project Settings -> API -> anon public key**
4. Run `flutter pub get`.
5. Start the app with `flutter run`.

The app throws a clear startup error if `.env` still has placeholder values.

## Required Tables

Create these tables in Supabase SQL editor:

```sql
create table if not exists people (
  id text primary key,
  first_name text not null,
  last_name text not null
);

create table if not exists checkins (
  id bigint generated by default as identity primary key,
  person_id text not null references people (id) on delete cascade,
  first_name text not null,
  last_name text not null,
  timestamp timestamptz not null default now(),
  notes text not null default ''
);
```
